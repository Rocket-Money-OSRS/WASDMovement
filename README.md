# WASD Movement Plugin

The WASD Movement Plugin allows you to control your RuneScape character with the WASD keys by simulating minimap clicks. This offers an intuitive and convenient way to move your character using familiar keyboard controls.

## Features

- Move your character using the W, A, S, and D keys
- Works in both fixed and resizable screen modes
- Intelligent minimap clicking for smooth player movement
- Stops the player when keys are released


## IMPORTANT
- Currently doesnt work with stretched mode plugin
- Has to be toggled off when typing currently - have a planned fix

## How It Works

The plugin listens for WASD key presses and simulates minimap clicks in the corresponding direction, effectively moving your character as if you clicked on the minimap.


## Usage

- Press and hold **W** to move up.
- Press and hold **A** to move left.
- Press and hold **S** to move down.
- Press and hold **D** to move right.
- Release all keys to stop moving.

## Configuration

No configuration is required for default use, but future options may be available under plugin settings.
